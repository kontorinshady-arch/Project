# Анализ unit - экономики развлекательного приложения 

Заказчик - приложение Procrastinate Pro+

Цель проекта - на основе данных, предоставленных компанией, необходимо провести анализ и ответить на вопросы:

- откуда приходят пользователи и какими устройствами они пользуются?
- сколько стоит привлечение пользователей из различных рекламных каналов?
- сколько денег приносит каждый клиент?
- когда расходы на привлечение клиента окупаются?
- какие факторы мешают привлечению клиентов?
Входные данные

Ход исследования:

- Обзор и предобработка данных;
- Исследовательский анализ данных;
- Анализ маркетинговых расходов;
- Оценка окупаемости рекламы.

Стек: 

pandas, numpy, matplotlib, datetime, LTV, CAC, retention rate.
